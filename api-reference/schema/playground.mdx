Playground is an interactive way to explore the schema and execute queries and mutations. It is available at the [https://dashboard.lightseries.io/api/graphql](https://dashboard.lightseries.io/api/graphql) endpoint of your API.

_**Don't forget to add the required authorization headers when making requests to the API.**_
